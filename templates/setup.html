重要说明

这是一个 Python Flask 后端应用，需要部署在支持 Python 的服务器上才能运行。 以下是部署步骤，请按照指引完成部署。

部署步骤

下载代码包
点击下方下载按钮，获取完整的项目代码压缩包。
上传到服务器
将代码上传到你的 Python 服务器（如阿里云、腾讯云、VPS 等）。
安装依赖
在服务器上运行以下命令安装依赖：
pip install -r requirements.txt
启动服务
运行以下命令启动应用：
python app.py
访问管理后台
启动后访问 http://你的服务器IP:5000 即可使用管理界面。

项目结构

wecom-assistant/ ├── app.py # 主应用入口 ├── config.py # 配置文件 ├── requirements.txt # Python 依赖 ├── services/ │ ├── wecom_service.py # WeCom API 服务 │ ├── analyzer_service.py # 消息分析服务 │ ├── scheduler_service.py # 定时任务服务 │ └── storage_service.py # 数据存储服务 └── templates/ └── index.html # Web 管理界面

下一步建议

  • 将代码部署到云服务器（阿里云、腾讯云、Railway、Render 等）
  • 配置系统服务实现开机自启
  • 设置 Nginx 反向代理并配置 HTTPS
  • 配置定时任务确保服务持续运行
  • 在管理界面设定你的目标，开始使用
